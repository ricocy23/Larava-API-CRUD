<h1>Introduction</h1>
Make API using Laravel and postman

## Project Setup

```
composer install
```

```
rename .env.example to .env
```

```
php artisan key:gen
```

<h2>Using 2 port</h2>

```
php artisan serve
for access http://localhost:8000/api/buku (backend)

```

```
php artisan serve --port=8001
for access http://localhost:8001/buku (frontend)
```
